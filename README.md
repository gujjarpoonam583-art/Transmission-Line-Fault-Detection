# ⚡ Transmission Line Fault Detection and Classification using MATLAB/Simulink

## 📌 Overview
This project focuses on the simulation and analysis of transmission line faults using MATLAB/Simulink. It demonstrates how different fault conditions affect voltage and current waveforms and explains the importance of protection systems in electrical power networks.

## 🎯 Objectives
- Simulate a three-phase transmission line.
- Detect and classify different fault types.
- Analyze voltage and current waveforms.
- Study power system protection concepts.

## ⚡ Fault Types
- Single Line-to-Ground (LG)
- Line-to-Line (LL)
- Double Line-to-Ground (LLG)
- Three-Phase Fault (LLL)

## 🛠️ Tools Used
- MATLAB
- Simulink
- Simscape Electrical
